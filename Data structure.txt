Data structure (v0.1):

Header: (10 bytes)
Byte  (1): ^ (Start of block marker, 0x5E = 94)
Long  (8): Enabled time (ms)
Byte  (1): Bitmask field: (X X X X X Groundtruth Drive Semaphore)

Semaphore: (8 bytes)
Long  (8): Last update time

Drive: (24 bytes)
4x:
Byte  (1): Output level
Byte  (1): Current
Byte  (1): Bus Voltage (*10)

Int   (4): Loops since last dump
Long  (8): Last update time
